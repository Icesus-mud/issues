# Icesus MUD — Bug Reports & Feature Requests

Report bugs, typos, and ideas for [Icesus MUD](https://icesus.org).

## How to report

**In-game (easiest):**
```
bug The shop in Vaerlon crashes when selling arrows
bug typo Room description says "teh" instead of "the"
bug idea Add fishing to the lake
```

**On GitHub:**
[Create a new issue](https://github.com/Icesus-mud/issues/issues/new) with a description. Include where you were in the game and what happened.

**On Discord:**
Report in the #bugs channel: https://discord.gg/j9cSPyAzQb

## What happens next

Wizards review reports and fix issues. You can track progress here on GitHub.

## About Icesus

Icesus is a free multiplayer text adventure (MUD) set on a frozen planet, online since 1995. Play at [icesus.org](https://icesus.org) or directly in your browser at [play.icesus.org](http://89.167.95.12).
